# Sofrología México

Sitio web profesional para **Sofrología México — Instituto SOS PHREN LOGOS®**.

## Cómo usarlo

1. Subir este proyecto a GitHub.
2. Conectar el repositorio con Vercel.
3. Publicar el sitio.
4. Conectar el dominio `sofrologiamexico.mx`.

## Comandos

```bash
npm install
npm run dev
```

## Secciones incluidas

- Inicio / Hero
- Servicios: niños, adolescentes y adultos
- Formación profesional
- Beneficios
- Testimonios
- Blog
- Agenda
- Contacto
